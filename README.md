## 简介

> 本代码来源于网络, 请入群(123300273)后联系群主索要sql文件!
>
> 前后分离, 前端springboot 后端 nodejs-angular,对技术要求比较高
> npm 运行:
> (1).删除package-lock.json文件,然后运行npm install
> (2).运行: npm run start
> (3).浏览器访问:  http://localhost:8080
